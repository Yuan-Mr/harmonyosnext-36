1. 动态二维码的坑与解决方案：
// 定时刷新逻辑
aboutToAppear() {
  this.timer = setInterval(() => {
    this.getNewQRCode() // 调用API更新
  }, 60000)
}

// 必须记得清除！
aboutToDisappear() {
  clearInterval(this.timer)
}
避坑指南：大屏折叠时记得监听断点变化，否则会出现布局错乱哦！
